World University Rankings

Interactive Visualization using D3 - Radar Chart

Nia Vivek and Shannon Hamilton

This visualization displays data from the Times Higher Education 2011 + 2016 World University Rankings. Select the universities from the drop-down menu to compare them.

Note: Null values for Income, Staff to Student Ratio, and International Score in the data were given the average value of that dimension.
